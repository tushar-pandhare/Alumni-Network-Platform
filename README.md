# Alumni Network Platform

Alumni Network Platform is a full-stack MERN application that connects students and alumni through mentorship, networking, career opportunities, events, startup collaboration, and community engagement.

## Features

* Authentication with Firebase & JWT
* Alumni Directory
* Mentorship Hub
* Job Board
* Events Management
* Startup Zone
* Alumni Stories
* Donation Portal
* Feedback Form
* Cloudinary Image Uploads
* Responsive UI

## Tech Stack

**Frontend**

* React.js
* Vite
* Tailwind CSS
* React Router
* Axios

**Backend**

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT

**Services**

* Firebase Authentication
* Cloudinary

## Installation

```bash
git clone https://github.com/tushar-pandhare/Alumni-Network-Platform.git

cd backend
npm install

cd ../frontend
npm install
```

## Configuration

Create `.env` files from the provided `.env.example` files in both frontend and backend directories.

## Run Locally

```bash
# Backend
cd backend
npm run dev

# Frontend
cd frontend
npm run dev
```

Open `http://localhost:5173`
